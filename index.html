Configuración

Ingresá tu clave API de Anthropic para usar el asistente. Tu clave se guarda localmente en este dispositivo y nunca se envía a ningún servidor excepto a Anthropic directamente.

Obtené tu clave en console.anthropic.com → API Keys. Si esta app está deployada en Vercel con la variable de entorno configurada, no necesitás ingresar la clave aquí.
